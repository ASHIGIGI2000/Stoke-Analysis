# Stoke-Analysis

Project Overview

This project analyzes a healthcare dataset to understand the factors that influence stroke risk. The data includes patient demographics, health conditions, and lifestyle information. The analysis helps identify patterns and visualize important insights using Microsoft Power BI.

Dataset

Stroke Prediction Dataset:

https://www.kaggle.com/datasets/fedesoriano/stroke-prediction-dataset

The dataset contains information such as:

Age
Gender
BMI
Glucose Level
Smoking Status
Heart Disease
Hypertension
Work Type
Stroke Status
Data Preparation

The dataset was cleaned and prepared before analysis.

Steps included:

Handling missing values in BMI and other columns
Replacing missing categorical values with "Unknown"
Fixing inconsistent text values
Removing duplicate records
Creating categories for Age Group, BMI Category, and Glucose Level
Formatting columns for proper data types

The cleaned dataset was saved and used for visualization.

Data Analysis in Power BI

The cleaned dataset was imported into Power BI and transformed to create useful metrics and categories.

Key steps included:

Verifying correct data types
Creating a Stroke Status column to classify stroke and non-stroke cases
Creating hierarchies for better analysis:
Age Group → Gender
Stroke vs No Stroke
Dashboard

An interactive Power BI dashboard was created to visualize stroke risk factors.

The dashboard includes:

KPI cards for Total Patients, Stroke Cases, and Stroke Rate
Charts showing stroke distribution by Age, Gender, and Marital Status
Health analysis using Hypertension, Heart Disease, BMI, and Glucose levels
Lifestyle analysis including Smoking and Work Type
Location analysis comparing Urban and Rural areas
Interactive filters for better exploration of the data
Key Insights
Stroke risk increases significantly with age
Hypertension and heart disease increase stroke risk
Higher BMI and glucose levels are linked to higher stroke cases
Smoking is an important lifestyle factor affecting stroke
Stroke occurrence is slightly higher in urban areas

Microsoft Power BI

Data Cleaning

Conclusion

This project demonstrates how healthcare data can be cleaned, analyzed, and visualized to identify important stroke risk factors. The dashboard provides insights that can support data-driven healthcare decision making.
